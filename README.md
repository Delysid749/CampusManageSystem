# CampusManageSystem

#### 介绍
社团管理系统

#### 软件架构
前端使用Vue+ElementUI，后端使用SpringBoot框架的社团管理系统
前端项目链接：https://gitee.com/Akaisuici/campus-vue


#### 主要功能

1.社团完成活动申请的具体信息并上交
2.学校完成对活动申请的审核，并反馈结果给社团
3.社团可以通过系统进行人员管理
4.社团可以通过系统发布公告
5.开发留言板块接受游客的意见
